========================
  Analysis core modules
========================

The core modules contain the important classes :class:`Simulation` and
:class:`Plugin`. An analysis class is derived from :class:`Simulation`
and additional plugins from :mod:`gromacs.analysis.plugins` are added
to the instance; these plugin classes must be derived from
:class:`Plugin`.
