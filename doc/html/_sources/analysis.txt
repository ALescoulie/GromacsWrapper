==========
 Analysis
==========

The analysis package uses the :mod:`gromacs` package and various other 3rd
party ones such as `numpy`_ and `pylab`_. It provides a frame work to
analyze `Gromacs`_ MD simulations.

.. _numpy: http://numpy.scipy.org
.. _pylab: http://pylab.scipy.org
.. _Gromacs: http://www.gromacs.org
